---
title: "Customize the Ticket Layout"
---
Page layout builder helps you to create custom layouts using a visual drag-drop editor. It also enables you to build different Ticket layouts based on user roles, teams, or personal preferences.

<Note>

**Note:** A User with **Manage Permission** Rights can perform this action

</Note>

To Customize the Ticket Layout,

- Navigate to the **Profile Icon** in the top right corner.
- Click on **Set Up**
- Head over to the **Modules** category
- Select **Tickets** 

<img src="images/009-ticket-module-setup-navigation-31916335335833.png" alt="Ticket module setup navigation" />
* Click on the **Customize the Page Layout option** * Here, you can * Create a
New Layout * Customize the Default Page Layouts

###  Create a New Layout

To create a New Page Layout,

- Click on the **Create New Layout** option on the right

<img src="images/010-create-new-ticket-layout-31916863770265.png" alt="Create new ticket layout" />
* A pop will open asking for the following details * **Layout Name**: Enter the
layout Name * **Choose Layout Frame**: Select the appropriate Layout Frame from
* 3 Column Layout - Default Layout * 2 Column Layout - Left Sidebar * 2 Column
Layout - Right Sidebar * 2 Column Layout - Equal width * Once you have provided
the information, click the **Start Customization** button

<img src="images/011-new-ticket-layout-details-31916928181785.png" alt="New ticket layout details" />
* You will be redirected to the **Visual Layout Builder** Page * Here, a blank
canvas will open with the selected layout frame and one action bar at the top.
You can drag-drop the required widget on the screen to create a preferred
layout.

<img src="images/012-ticket-layout-builder-page-31916972479641.png" alt="Ticket layout builder page" />
* Build different **Accordions** and **Tabs** to organize the content in a
specific structure.

<img src="images/013-ticket-layout-accordions-and-tabs-31917479078681.png" alt="Ticket layout accordions and tabs" />
* When accessing the Ticket Record Details, you have the flexibility to
customize your view. You can choose to display either All Available Fields or
only specific ones by selecting the **"Add Field"** option

<img src="images/014-add-field-to-ticket-layout-31918909396889.png" alt="Add field to ticket layout" />
* You can also **Set the Visibility Rule** from the **\+ Add Rule** option *
Set the **Condition Criteria and Save** 

<img src="images/015-set-visibility-rule-to-ticket-layout-31919135095321.png" alt="Set visibility rule to ticket layout" />
* Once you are done making the changes, you can hit on **Save**.

<img src="images/016-save-ticket-layout-31919135096857.png" alt="Save ticket layout" />
* A pop will open asking for the following information, * **Layout Name**: It
will display the name that you provided earlier, but you can change it before
saving. * **Description**: Add a description of the newly created layout. *
Choose if you want to Enable this layout for usage. If you enable the option it
will open an option to allow access, * **Organization Level**: Everyone in the
organization will have access to this layout. * **Role**: Only users with
specific roles can access this layout.

- **Teams**: Only users in the specific team can access this layout.

- **User**: Only specific users can access this layout.
- Then, **Save** the information

<img src="images/017-new-ticket-layout-details-save-31919240227097.png" alt="New ticket layout details save" />

<Note>

**Note:** If you have one layout set as organization level and then you save
 another then the previous one loses its org level setting and becomes inactive
 from usage at user level.

</Note>

###  Customize the Default Layouts

To customize the Default Layout

- Hover on the **Default Layout** option,**Actions** button will appear
- Click on the **Actions** button and select the **Edit** option.

<img src="images/018-edit-default-ticket-layout-31919643355033.png" alt="Edit default ticket layout" />
* You will be redirected to the **Layout Builder Page**. * Here, you can drag
and drop the Widgets to align with your specific business processes * Make the
desire changes, once done click on **Save**

<img src="images/019-ticket-layout-builder-page-31919860027801.png" alt="Ticket layout builder page" />
* Once you click on the Save option a pop will open asking for the following
information * Here, you can * **Change the Layout Name** * **Add the
Description** * **Enable/Disable the Layout for Usage** * **Change the Layout
Access** * Once done, **Save** the information.

<img src="images/020-edit-default-ticket-layout-details-32099045399705.png" alt="Edit default ticket layout details" />
