### How to access Smart Flow settings

To access your Smart Flow Settings,

Click on a Smart Flow TitleHover the cursor towards the arrow visible besides the settings icon on left side menu bar under the flow title

### Enrollment Conditions:

Using this setting, you can configure your record's enrollment condition. The available options are:**Only Once**: This can be used if you do not want a record to be enrolled more than once in a flow**More than Once But Single Active Instance:**This should be used when you want a record to be enrolled if it is not active i.e. not at any stage of the specific flow.**More Than Once Whenever the Conditions are met:**This should be used if you want a record to be enrolled multiple times in a flow disregarding if it is already active i.e. Sending a meeting booking confirmation every time a customer books a meeting

![Screen Shot 2025-03-07 at 4.24.37 PM.png](./images/Screen-Shot-2025-03-07-at-4.24.37-PM.png)

![Screen Shot 2025-03-07 at 4.26.49 PM.png](./images/Screen-Shot-2025-03-07-at-4.26.49-PM.png)

If you are making any changes to the existing enrollment condition, please make sure to hit**"Save"**"**Publish"**so that the new enrollment condition takes effect.

![Screen Shot 2025-03-07 at 4.28.29 PM.png](./images/Screen-Shot-2025-03-07-at-4.28.29-PM.png)

### Flow Manager

Flow Manager should be used to understand and manage the elements used in a given flow. Elements are:

![Screen Shot 2025-03-07 at 4.36.05 PM.png](./images/Screen-Shot-2025-03-07-at-4.36.05-PM.png)

**Variables**: This allows you to view/create new variable for a given flow

![Screen Shot 2025-03-07 at 4.30.59 PM.png](./images/Screen-Shot-2025-03-07-at-4.30.59-PM.png)

**Actions:**This lets you view number of actions and what the are the actions used in a flow

![Screen Shot 2025-03-07 at 4.33.43 PM.png](./images/Screen-Shot-2025-03-07-at-4.33.43-PM.png)

**Conditions:**This lets you view the number of condition nodes and what the are the conditions used in the flow.

![Screen Shot 2025-03-07 at 4.34.23 PM.png](./images/Screen-Shot-2025-03-07-at-4.34.23-PM.png)
