When a Flow is created and remains in draft status, Salesmate provides the flexibility to modify the trigger if needed. This allows you to adjust the starting conditions of the flow to better align with changing requirements, without needing to create a new flow entirely.


### How to Change the Flow Trigger

To change the Flow Trigger,


*   Access**Smart Flows**feature from left menu bar.

![](./How to Change the Flow Trigger – Salesmate_files/mceclip0.png)


*   Select the desired Flow where you wish to change the Trigger.

![](./How to Change the Flow Trigger – Salesmate_files/mceclip0(1).png)


*   Hover over the trigger, click on the three dots, and select Change.

![](./How to Change the Flow Trigger – Salesmate_files/mceclip1.png)


*   A popup will show a list of available Triggers, select the new trigger from the list

![](./How to Change the Flow Trigger – Salesmate_files/mceclip2.png)


*   Once done, click on**Save**to save the changes
