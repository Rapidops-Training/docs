Manual Flow initiates when a user manually publishes and enrolls a record. If you have any active Manual Flows linked, you can enroll records from both the record detail and listing pages.**Topics Covered:**[How to Create Manual Flow](https://support.salesmate.io/hc/en-us/articles/37578474205081-Working-with-Manual-Flows#h_01JBY8JK3DSKGCVM5K6YFYF0WF)[How to Enroll Records to Manual Flow](https://support.salesmate.io/hc/en-us/articles/37578474205081-Working-with-Manual-Flows#h_01JB8TJT0AW30BHQ1WAEKVRC3A)

### How to Create Manual Flow

To create the Manual flow,

Navigate to**Smart Flows**from left side outreachClick on**New Flow****

![mceclip1.png](./images/mceclip1-5.png)

**Select the flow type as**Manual Flow****

![Screen Shot 2024-11-15 at 2.14.37 AM.png](./images/Screen-Shot-2024-11-15-at-2.14.37-AM.png)

**For**Manual Flow**, you'll need to choose the record module

![mceclip2.png](./images/mceclip2-5.png)

Proceed with selecting Actions and Conditions to complete the flowOnce the Flow is created you can go ahead and**Save and Publish**the Flow.**Note:**While creating a manual flow, there is no option to create the trigger. You have to select the module and then start with actions and conditions.

### How to Enroll Records to Manual Flow

There are two ways to enroll the records,

[From the Detail View](https://support.salesmate.io/hc/en-us/articles/37578474205081-Working-with-Manual-Flows#01JBY8CWVD41PQYHY7VBDCBZBY)[From the List View](https://support.salesmate.io/hc/en-us/articles/37578474205081-Working-with-Manual-Flows#h_01JBVQ62RC2VR7WQ13SW4QVNFC)

#### From the Detail View

Navigate to the required**Module**from the left menu barOpen the**Detail view**of the specific record you wish to enroll.Click on the**Three dots**in the top right cornerSelect the**Enroll to Flow**option.

![mceclip0.png](./images/mceclip0-9.png)

A popup will appear, showing the available manual flows for enrollment.

![mceclip1(1).png](./images/mceclip11-2.png)

Select the desired manual flows from the list.Click the**Enroll**button to finalize your selection.

![mceclip3.png](./images/mceclip3-3.png)

A success message will be displayed:**Record(s) added to enrollment queue.**#### From the List View

Navigate to the Listing screen of the required Module.Select the records you want to enroll in by checking the corresponding boxes.Click on the**Enroll to Flow**option in the bulk actions toolbar.

![mceclip4.png](./images/mceclip4-2.png)

A popup will appear, displaying the available manual flows for enrollment.

![mceclip5.png](./images/mceclip5.png)

Select the desired manual flows from the list.Click the Enroll button to confirm your selection.

![mceclip6.png](./images/mceclip6-2.png)

A success message will appear:**Record(s) added to enrollment queue.**A manual flow can have the following enrollment conditions:**Only once:**A record can only enter this flow once**More than once, but single active instance:**A record can enroll in this flow whenever the conditions are met, provided that they are not running through the same flow.**More than once, whenever the conditions are met:**A record can enter this flow each time the trigger conditions are met.
