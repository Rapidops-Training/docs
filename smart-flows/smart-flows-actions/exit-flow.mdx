The Exit Flow Action allows you to remove a record from the current Smart Flow or all active flows. This action is useful when you need to stop automated processes for specific records.Topics Covered:

[How to Configure Exit Flow Action](https://support.salesmate.io/hc/en-us/articles/38138458679577-Exit-Flow#h_01J9A1AA74WA2E18G52SW877W7)[Practical Example](https://support.salesmate.io/hc/en-us/articles/38138458679577-Exit-Flow#h_01J9A1APSPVKW60HRC7S5A232P)

### How to Configure Exit Flow Action

When setting up a Smart Flow, select the**Exit Flow**action.

![Screen Shot 2024-11-11 at 1.03.28 PM.png](./images/Screen-Shot-2024-11-11-at-1.03.28-PM.png)

When setting up the Exit Flow Action, you will need to configure the following details:**Name**: Provide a clear and descriptive name to identify the action's purpose.**Description**: Give a brief explanation of the action's function, describing what it will achieve.**Exit Instance(s) Selection**: You can choose**Exit from Current Flow**to remove the recordâ€™s current instance from the flow (default option) or choose**Exit from all Flows**to exit the record from all flows, which will stop the automation for that record across all workflows.

![Screen Shot 2024-11-11 at 1.06.07 PM.png](./images/Screen-Shot-2024-11-11-at-1.06.07-PM.png)

Once the configuration is completed, hit**Save**.

### [Practical Example](https://support.salesmate.io/hc/en-us/articles/38138458679577-Exit-Flow#h_01J9A1APSPVKW60HRC7S5A232P)

The exit flow action allows you to remove a contact from the current flow or even from all flows altogether. For instance, if a client unsubscribes from an email, you can use this action to remove them from the flows since they are no longer interested.
