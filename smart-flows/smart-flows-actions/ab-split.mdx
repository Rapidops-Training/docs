### How to Configure A/B Split Action


*   When setting up a Smart Flow, select the**A/B Split**Action.

![Screen Shot 2024-11-09 at 9.56.54 PM.png](./A_B Split – Salesmate_files/Screen Shot 2024-11-09 at 9.56.54 PM.png)

To configure the**A/B Split action,**you’ll need to enter the following details:


***Name**: Assign a clear and descriptive name for the A/B Split Action to easily identify its purpose.


***Description**: Provide a brief explanation of the action’s intent.

    ***Percentage for Path A**: Define the percentage of participants or records that should follow Path A.
    ***Percentage for Path B**: Set the percentage of participants or records that should follow Path B, ensuring the total equals 100%.

*   Once the configuration is completed, hit on**Save**.
