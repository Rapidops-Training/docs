Salesmate provides an option to Set Up the new Subscription type in your preferred language.To set up a new Subscription Type in different languages,

Navigate to your**Profile Icon**on the top right cornerClick on**Setup**Head over to**Email & Delivery**Click on**Subscription Types**

![Screen Shot 2024-03-28 at 3.02.31 PM.png](./images/Screen-Shot-2024-03-28-at-3.02.31-PM-2.png)

Click on**\+ Subscription Type**

![Screen Shot 2024-03-29 at 11.34.14 AM.png](./images/Screen-Shot-2024-03-29-at-11.34.14-AM-2.png)

Here, you need to add,**Name:**Give a name to the subscription type**Consent Type:**Select Opt-In or Opt-Out [Click here to learn more about the Consent type](https://support.salesmate.io/hc/en-us/articles/19613366377241)**Language:**Select the language, by default it will be**English**To add a different language, click on**Add a Language Variant**

![Screen Shot 2024-03-29 at 2.22.19 PM.png](./images/Screen-Shot-2024-03-29-at-2.22.19-PM.png)

Once the language is selected the Title and Description will get auto-copied from the English language variant by default which can be updated by the User.Click on**Save**

![Screen Shot 2024-03-29 at 2.47.39 PM.png](./images/Screen-Shot-2024-03-29-at-2.47.39-PM.png)

If you wish to Remove the Language Variant, you can do so by clicking on the**Remove**option next to the Variant.

![Screen Shot 2024-03-29 at 2.48.24 PM.png](./images/Screen-Shot-2024-03-29-at-2.48.24-PM.png)

####**Note:**The default language cannot be changed.The same Language Variant cannot be used more than once
