## What is Unified Inbox?

Salesmate introduces a Unified Conversation Inbox, integrating chat conversations from various diverse social networks along with website chat. This centralized hub optimizes communication management, presenting a cohesive interface that seamlessly amalgamates messages from multiple channels. This enhancement is designed to significantly improve efficiency and convenience in handling diverse communication streams.

<Note>
  **Pricing**: Please Note that Unified Conversation Inbox is an add-on feature.
  It is priced at **`$15` per user per month** or **`$144` per user per year**
  (**`$12` effective monthly**).
</Note>
