Working with email templates can be a great way to save some time composing emails or answering repeated requests.

In Salesmate Files cannot be directly attached in email templates, but you get options to attach raw files while selecting the email template to send an email.

You can use Dropbox, Google Drive, or any other file hosting services for uploading common files to templates.
