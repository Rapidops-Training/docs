title: "How to Add Today's Date in an Email Template?"
description: "To add today's date in an email template, You can use special word \"today\" with date function."
---

To add today's date in an email template, You can use special word "today" with date function.

Add the following attribute in the email template:**Input**```
This page was last updated at {{ "today" | date: "%Y-%m-%d" }}
```**Output**```
This page was last updated at 2024-03-12
```

![](./How to Add Today's Date in an Email Template_ – Salesmate_files/mceclip0.png)

You can also add the current time with the date.**Input:**```
This page was last updated at {{ "today" | date: "%Y-%m-%d %H: %M" }}
```**Output:**```
This page was last updated at 2024-03-12 19:06
```
