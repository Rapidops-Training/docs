You can insert text functions in the Email Template to append, prepend, uppercase, downcase, concat, and remove texts from a particular field. Topics covered:

[Append](https://support.salesmate.io/hc/en-us/articles/360054932551-Inserting-Text-Functions-in-an-Email-Template#h_01H25SSGRXW31GPYA2589H6Q4T)[Prepend](https://support.salesmate.io/hc/en-us/articles/360054932551-Inserting-Text-Functions-in-an-Email-Template#h_01H25SST5QHM09RA3BJ41D2D0X)[Uppercase](https://support.salesmate.io/hc/en-us/articles/360054932551-Inserting-Text-Functions-in-an-Email-Template#h_01H25ST3MMSTXV3Z60DDTZ4ZM4)[Downcase](https://support.salesmate.io/hc/en-us/articles/360054932551-Inserting-Text-Functions-in-an-Email-Template#h_01H25STBF0Z5QW1X90VSS6PMDX)

### Append

This function appends the text mentioned by you along with the value of the field.**Input:**```highlight
Append Example :&#123;&#123;Contact.name | append:", Hope you are doing well"!&#125;&#125;
```

![Screenshot](./images/Screenshot-2023-06-05-at-3.29.44-PM.png)

**Output:**```text highlighter-rouge
Append Example: &#123;Lina Westminster, Hope you are doing well!&#125;
```

![Screenshot](./images/Screenshot-2023-06-05-at-3.31.02-PM.png)

### Prepend

This function prepends the text mentioned by you along with the value of the field.**Input:**```highlight
Prepend Example : &#123;&#123;Contact.name | prepend : "Hello"&#125;&#125;
```

![Screenshot](./images/Screenshot-2023-06-05-at-3.34.31-PM.png)

**Output:**```highlight
Prepend Example: Hello Lina Westminster
```

![Screenshot](./images/Screenshot-2023-06-05-at-3.40.28-PM.png)

### Uppercase

This function converts each character of the word in uppercase. Note:  It has no effect on strings that are already all uppercase.**Input:**```highlight
Uppercase : &#123;&#123;"Contact.name" | upcase&#125;&#125;
```

![mceclip4.png](./images/mceclip4-3.png)

**Output:**```highlight
Uppercase: ABIE HANBY
```

![mceclip3.png](./images/mceclip3-3.png)

### Downcase

This function converts each character of the word in lowercase.Note:  It has no effect on words that are already all lowercase.**Input:**```highlight
LowerCase: &#123;&#123;"Contact.name" | downcase&#125;&#125;
```

![mceclip0.png](./images/mceclip0-4.png)

**Output:**```text highlighter-rouge
LowerCase: abie hanby
```

![mceclip1.png](./images/mceclip1-5.png)
