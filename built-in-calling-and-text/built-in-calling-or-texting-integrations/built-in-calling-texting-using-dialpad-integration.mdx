**Available in the Pro Plan and above.**

Note: The user email id on Salesmate and DialPad should be the same then only it will allow you to send or receive a text in Salesmate.

[Dialpad](https://www.dialpad.com/) is the most modern business communications platform. Taking into account every form of communication that you rely on and unified it into one app. Calling a client? Meeting with your team? Texting a colleague? It’s all here, on all your devices.

Installing / Uninstalling the Dialpad app (Only for Admin)

- Navigate to the **Profile Icon** on the top right corner
- Click on **Setup**
- Head over to **Voice & Text** category
- Click on **Voice Apps**

![](./Built-in Calling & Texting using Dialpad Integration – Salesmate_files/mceclip0.png)

- **Dialpad**
- Click on “**Install**” to install the **Dialpad**

![Screen_Shot_2023-05-09_at_3.20.50_PM.png](./Built-in Calling & Texting using Dialpad Integration – Salesmate_files/Screen Shot 2023-05-09 at 3.20.50 PM.png)

- You will be asked to provide your **Dialpad key** here.
- **Dialpad API** keys can be generated from the native **Dialpad account**. [Create Dialpad API Key](https://help.dialpad.com/hc/en-us/articles/360019680532-Create-an-API-Key)

To create an API key, head to your **Admin Settings** from **Dialpad.com.**

- Navigate to **My Company**
- Select **Authentication**
- Select **API Keys**
- Select **Add key**
- Enter the API key's name and expiration terms
- Select desired Additional Scopes
  - For Call Sync (Recording export should be selected)
  - For SMS Sync (Message content export should be selected)

Recordings export

Allow recordings URL to be provided in call logs

Message content export

Allow text message content to be read for all users

Dialpad Meetings read

Allow read-access to Dialpad Meetings resources

Screen pop

Allow screen pops initiated from screen pop API

Change log

Allow subscription to change log events using event subscription API

![Screenshot](./Built-in Calling & Texting using Dialpad Integration – Salesmate_files/Screenshot 2023-06-27 at 3.21.20 PM.png)

- Select **Save**
- If desired, copy the code to your clipboard

![Screenshot](./Built-in Calling & Texting using Dialpad Integration – Salesmate_files/Screenshot 2023-06-27 at 3.25.40 PM.png)

Your new API Key and will now appear in the list of API keys.

- Add the Dialpad API Keys as created

![](./Built-in Calling & Texting using Dialpad Integration – Salesmate_files/mceclip0(1).png)

- On successful verification, the following message is shown in status: “**Integration enabled**”

![](./Built-in Calling & Texting using Dialpad Integration – Salesmate_files/mceclip1.png)

- On Failure: “**Connection failed. Please check your API keys**”.

![](./Built-in Calling & Texting using Dialpad Integration – Salesmate_files/mceclip2.png)

- Call and Text Logging

**\>> Log calls and texts of all the users even without a Salesmate account.**

Choose a user to assign calls and texts from unmatched users (**not available in Salesmate**). (Refer to [Calling via Dialpad](https://support.salesmate.io/hc/en-us/articles/9333549016857-Calling-via-Dialpad))

![Screenshot](./Built-in Calling & Texting using Dialpad Integration – Salesmate_files/Screenshot 2023-06-27 at 3.30.29 PM.png)

- **Log calls and texts of the users having both Salesmate and Dialpad accounts.** The users are auto matches based on their email.
- Log calls and texts of all Dialpad users (Default: Enabled)
- Log calls and texts of common users
- Click “**Save**”

**To uninstall the DialPad app.**

- Navigate to the **Profile Icon** on the top right corner
- Click on **Setup**
- Head over to **Voice & Text** category
- Click on **Voice Apps**

![](./Built-in Calling & Texting using Dialpad Integration – Salesmate_files/mceclip0.png)

- **Dialpad**
- Click on “**Remove**” to uninstall the **Dialpad  
  **
- You will get a confirmation popup.
- “Are you sure you want to remove the Dialpad App?
- Confirm to remove the Dialpad app

![Screen_Shot_2023-05-09_at_3.30.23_PM.png](./Built-in Calling & Texting using Dialpad Integration – Salesmate_files/Screen Shot 2023-05-09 at 3.30.23 PM.png)
