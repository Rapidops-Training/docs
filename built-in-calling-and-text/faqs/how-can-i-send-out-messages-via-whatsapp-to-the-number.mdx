---
title: "How can I send out messages via WhatsApp to the number?"
description: "Navigate to the calling configuration page by following the steps down below..."
---

Navigate to the calling configuration page by following the steps down below

*   Navigate to the **Profile Icon** on the top right corner
*   Click on **Setup**
*   Head over to **Voice & Text** category
*   Click on **Calling Configuration**

![](./How can I send out messages via WhatsApp to the number_ – Salesmate_files/mceclip0.png)

*   Now, add the provider field as **Whatsapp** and add the Syntax as [https://web.whatsapp.com/send?phone=\[number\]](https://web.whatsapp.com/send?phone=%5Bnumber%5D)

![](./How can I send out messages via WhatsApp to the number_ – Salesmate_files/mceclip1.png)

*   Once you have set up the configuration, you will be able to send out the messages using Whatsapp.

![](./How can I send out messages via WhatsApp to the number_ – Salesmate_files/mceclip2.png)
