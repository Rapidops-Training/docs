### Settings for Chrome

While you're using Salesmate in Chrome browser.

In the**address bar**, click the**padlock icon**. Navigate to '**Site settings**.'

![Screenshot_2023-05-31_at_12.58.06_PM.png](./images/mceclip0.png)

Then, make sure '**Sound**' is set to '**Allow**':

![Screenshot_2023-05-31_at_1.02.17_PM.png](./images/Screenshot 2023-05-31 at 1.02.17 PM.png)

![](./images/mceclip0.png)
