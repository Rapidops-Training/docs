Setting the "Required" rule to fields in Salesmate ensures that certain data must be provided before saving or submitting a form, helping maintain data completeness and accuracy.

<Note>
**Note:**Available for**Pro**and above Plans
</Note>


### How to Set Field Required Rule

To Set the Field Required Rule,


*   Navigate to the**Profile Icon**on the top right corner

*   Click on**Set Up***   Head Over to the**Modules Category***   Select the Module(**Contact, Company, Deal, Products, and Custom Modules)****![](./How to Set Field Required Rule – Salesmate_files/mceclip1.png)***   Click on**Customize the Create Module Form**under Layout

![](./How to Set Field Required Rule – Salesmate_files/mceclip2.png)


*   Choose the field for which you wish to Set the Required Rule, then click on the three dots located at the corner

*   Click on the**Set Required Rule**option

![](./How to Set Field Required Rule – Salesmate_files/mceclip0.png)

OR


*   Go to**All Detail Form Field Section***   To Set the Required Rule for a field, hover over the field you wish to modify and click on the**Actions**button.

*   Click on the**Set Required Rule**option

![](./How to Set Field Required Rule – Salesmate_files/mceclip1(1).png)


*   Set the Required Criteria

*   Once the condition is set, hit**Save**.

![](./How to Set Field Required Rule – Salesmate_files/mceclip2(1).png)

<Note>
**Note:**Please Note that a user with**Manage Modules**permission can configure these settings.
</Note>
