###  How to Set Field Required Rule

To Set the Field Required Rule,

Navigate to the **Profile Icon** on the top right cornerClick on **Set Up** Head Over to the **Modules Category** Select the Module(**Contact, Company, Deal, Products, and Custom Modules)***![mceclip1.png](./images/mceclip1-9.png)

- *Click on** Customize the Create Module Form ** under Layout

![mceclip2.png](./images/mceclip2-7.png)

Choose the field for which you wish to Set the Required Rule, then click on the three dots located at the cornerClick on the **Set Required Rule** option

![mceclip0.png](./images/mceclip0-6.png)

OR

Go to **All Detail Form Field Section** To Set the Required Rule for a field, hover over the field you wish to modify and click on the **Actions** button.Click on the **Set Required Rule** option

![mceclip1(1).png](./images/mceclip11-4.png)

Set the Required CriteriaOnce the condition is set, hit **Save**.

![mceclip2(1).png](./images/mceclip21-2.png)

**Note:**Please Note that a user with **Manage Modules** permission can configure these settings.
