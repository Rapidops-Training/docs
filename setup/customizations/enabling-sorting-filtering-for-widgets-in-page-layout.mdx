The Modules Page Layout Builder empowers you to personalize your Detail Layout experience by enabling sorting and filtering functionalities for the selected widgets.

<Note>
**Note:** Enable Sorting & Filtering is available for Associated Modules (Multi Look Up Fields), Associated Products, Deals & Activities widgets
</Note>

To enable Sorting Filtering for Widgets,


*   Navigate to the**Profile Icon**in the top right corner.

*   Click on**Set Up***   Head over to the**Modules**category

*   Select the required Module**(Contact, Company, Deal, Product or Custom Module)**![](./Enabling Sorting & Filtering for Widgets in Page Layout – Salesmate_files/mceclip0.png)


*   Click on the**Customize the Page Layout**option

![](./Enabling Sorting & Filtering for Widgets in Page Layout – Salesmate_files/mceclip1.png)


*   Here, you can Edit the existing Layout or Create a New Layout

![](./Enabling Sorting & Filtering for Widgets in Page Layout – Salesmate_files/mceclip2.png)


*   When you drag-drop the Associated Module widget on the layout page, a drop-down will ask you to choose the relation.

*   Once you have selected the module, an option to add the fields will be enabled.

![](./Enabling Sorting & Filtering for Widgets in Page Layout – Salesmate_files/mceclip4.png)


*   Now if you add any of the above mentioned Widgets, you would get an option to,
    *   [Enable Sorting](https://support.salesmate.io/hc/en-us/articles/35196433650713-Enabling-Sorting-Filtering-for-Widgets-in-Page-Layout#h_01J32NP6NH13ZES65H3S5TGC5A)
    *   [Apply Filters](https://support.salesmate.io/hc/en-us/articles/35196433650713-Enabling-Sorting-Filtering-for-Widgets-in-Page-Layout#h_01J32Q58AFPXP364X70K9KR5MV)

![](./Enabling Sorting & Filtering for Widgets in Page Layout – Salesmate_files/mceclip3.png)


### Enable Sorting


*   This option will be disabled by default, click on the Toggle button to enable the option.

*   Choose the field on which you want to apply sorting and then select the order of sorting,
    ***Select Field:**Select the required field from the available list of fields.
    ***Sort Order:**Choose to sort the order in ascending or descending order.

![](./Enabling Sorting & Filtering for Widgets in Page Layout – Salesmate_files/mceclip5.png)


### Apply Filters


*   Click on**+ Add Filter**to apply Filters

*   A pop up will open to add the set of Conditions

*   Once done,**Save**the applied criteria.

![](./Enabling Sorting & Filtering for Widgets in Page Layout – Salesmate_files/mceclip6.png)


*   Click on**Save**to save the applied changes.

![](./Enabling Sorting & Filtering for Widgets in Page Layout – Salesmate_files/mceclip8.png)
