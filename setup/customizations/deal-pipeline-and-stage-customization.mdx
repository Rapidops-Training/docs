Salesmate offers you a default Sales Pipeline with Stages. The user can, however, rename Stages and Pipeline according to their need.

Easy steps to configure your Sales Pipeline**Topics Covered:***   [How to Create a New Pipeline](https://support.salesmate.io/hc/en-us/articles/115000147089#h_01HQTBF6CN4FKR3ZV6H004GC39)

*   [Update the Default Pipeline](https://support.salesmate.io/hc/en-us/articles/115000147089#h_01HQTCBVRJHRJWR9F59WG88WY2)

*   [How to Delete a Pipeline](https://support.salesmate.io/hc/en-us/articles/115000147089#h_01HQTERWX20QBVF2NJN97KGEAT)


### How to Create a New Pipeline,

To create a New Pipeline,


*   Navigate to the**Profile icon**in the top right corner

*   Click on the**Setup***   Head over to the**Modules**category

*   Click on the**Pipeline**Section

![](./Deal Pipeline and Stage Customization – Salesmate_files/mceclip0.png)


*   Here, you need to enter the following details,
    ***Pipeline Name**: Define the Pipeline Name
    ***Allow users to set the probability for each deal manually**: Tick the checkbox if you wish to allow other Users to Set the Win Probability
    *   You can**Customize the Default Stage Colour, Rename them, Set the Win Probability and the Rotting period.***   You can also**Add a New Stag**e if required or just Remove it if not needed.
    *   rearrange the Stages with simple drag-and-drop
    *   Once done click on**Save**![](./Deal Pipeline and Stage Customization – Salesmate_files/mceclip4.png)


### Update the Default Pipeline

To Update the Default Pipeline,


*   Click on the Edit option of the Default

![](./Deal Pipeline and Stage Customization – Salesmate_files/mceclip3.png)


*   Here, you can Customize the Pipeline as per your requirement

![](./Deal Pipeline and Stage Customization – Salesmate_files/mceclip0(1).png)


### How to Delete a Pipeline


*   Click on the**Delete**icon to delete the Pipeline

![](./Deal Pipeline and Stage Customization – Salesmate_files/mceclip1.png)


*   Confirm the action to continue deleting the Pipeline
