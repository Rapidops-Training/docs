### How to Set Field Visibility Rule

To Set the Field Visibility Rule,

Navigate to the**Profile Icon**on the top right cornerClick on the**Set Up**Head Over to the**Modules**CategorySelect the Module(**Contact, Company, Deal, Products, and Custom Modules)**

![mceclip1.png](./images/mceclip1-10.png)

Click on**Customize the Create Module Form**under Layout

![mceclip2.png](./images/mceclip2-8.png)

Choose the field for which you wish to Set the Visibility Rule, then click on the three dots located at the cornerClick on the**Set Visibility Rule**option

![mceclip0.png](./images/mceclip0-7.png)

OR

Go to**All Detail Form Field Section**To set the visibility for a field, hover over the field you wish to modify and click on the**Actions**button.Click on the**Set Visibility Rule**option

![blobid0.png](./images/blobid0-2.png)

Set the Visibility CriteriaOnce the condition is set, hit**Save**.

![mceclip2(1).png](./images/mceclip21-3.png)

**Note:**Please Note that a user with**Manage Modules**permission can configure these settings.
