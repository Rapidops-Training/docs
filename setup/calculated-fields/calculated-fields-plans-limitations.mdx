Calculated fields allow you to perform automatic calculations based on existing data from other fields or related modules.


### Plans & Pricing

<Note>

Note:


*   Calculated fields are available only in**Pro and above**plans

*   They are**not available**for use in**legacy plans**.

</Note>

| Plans | No of Calculated Fields |
| --- | --- |
| Basic | - |
| Pro | 3 Per Module |
| Business | 20 Per Module |
| Enterprise | 50 Per Module |
