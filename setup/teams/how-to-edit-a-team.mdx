**Note**: Only users with an "Admin" role can perform this action.

### To Edit a Team :

Navigate to the**Profile icon**on the top right cornerClick on**Setup**Head over to**Users and Security**Click on**Teams**

![mceclip0.png](./images/mceclip0-4.png)

You will find a list of already-created teamsClick on**Edit**to change/update the field values.

![mceclip2.png](./images/mceclip2-3.png)

You can update the fields as shown below.Click on**Update**to save the changes.

![mceclip3.png](./images/mceclip3-2.png)
