<Note>

- **Note:** Only users with ‘Manage Messenger Settings’ access can configure these settings.

</Note>

Now add Salesmate Chat to your mobile apps directly and provide your customers an in-app chat experience.

- To install Chats to your Website, click here.
- To install chats to your iOS app, [click here](https://support.salesmate.io/hc/en-us/articles/7972525508633).

Follow the below steps to navigate to the Chat installation page:

- Navigate to **Conversations Icon** on the left menu bar
- Click on **Chats**- Head to **Inbox** **Settings** on bottom left.

<img src="images/009-open-chat-settings-50769976591769.png" alt="Open Chat settings" />
- Here, click on **Channel** and then locate the **Salesmate Chat** channel.
From the **Actions** menu, select **Edit**.

<img src="images/010-open-chat-settings-50769976595865.png" alt="Open Chat settings" />
- Here, click on Installation and you will get 3 tabs Web, iOS, and Android,
select the **Android** tab. - Installation of Chat on your Android app helps
sync live user data from your Android app to Salesmate, and lets you send in-app
messages.

<img src="images/011-open-chat-settings-50769976597657.png" alt="Open Chat settings" />

### Installing:

The android installation screen has two sections

- [**API Key**](#api-key-you-can-use-the-below-keys-to-add-the-salesmate-chats-to-your-app)

- [**Push Notifications**](https://support.salesmate.io/hc/en-us/articles/7972447855001-Installing-Chats-on-your-Android-App#h_01G6SNM99BDKBFHXCR4KY67MNA)

### - **API Key:** You can use the below keys to add the Salesmate Chats to your app.

- **API Key:** A unique key for your Salesmate account.

- **Workspace ID:** A unique key for your Salesmate Chat workspace

- **Tenant ID:** This is your Salesmate domain link.

<img src="images/012-open-chat-settings-50769987553817.png" alt="Open Chat settings" />
- The code is available in two languages

- **Java**-**Kotlin**
<img src="images/013-open-chat-settings-50769987555225.png" alt="Open Chat settings" />
####- **Enable Push Notifications:** - To send push notifications through
Messenger to your Android device, you need to configure the below settings. -
The push notifications are sent for newly received messages on Salesmate
Messenger when visitor/contact leaves the app.

Here, you need to add the below details to the Salesmate Chat settings

- **Server API Key:** Add your server API key for Firebase Cloud Messaging

- **Firebase service account file (Only JSON file is supported)**Once, you add the above information, the **Save** button would be enabled.
